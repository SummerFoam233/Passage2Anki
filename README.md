# Passage2Anki
## 功能介绍

以指定的单词考纲为范围，自动从文章中搜寻在考纲中出现的单词的所在句子，翻译后添加到Anki牌组中，实现在特定语境下理解单词含义，方便阅读后复盘。

## 效果展示

### 主界面

填写个人有道翻译的App ID和App Key，以及考纲（限定txt格式，一行一个单词）目录，和需要处理的文章文本。

### 结果界面

以表格形式呈现得到的单词结果，并自行选择是否要加入到牌组中。
